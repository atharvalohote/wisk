## ğŸš€ Getting Started

1. **Clone the repo:**
   ```sh
   git clone https://github.com/atharvalohote/wisk.git
   cd wisk
   ```
2. **Install dependencies:**
   ```sh
   npm install
   # or
   yarn install
   ```
3. **Set up API keys:**
   - Add your Google Gemini and Vision API keys in `app.json` under `extra`.
4. **Run the app:**
   ```sh
   npx expo start
   ```
   Scan the QR code with Expo Go or run on an emulator.
